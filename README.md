# Proyecto Frontend - Fullstack Node.js (Codo a Codo)

## Descripción

Este proyecto es el frontend del proyecto integrador que permite gestionar profesores, alumnos, cursos e inscripciones. Está desarrollado utilizando React y Vite.

## Scripts

- `dev`: Inicia la aplicación en modo desarrollo.

- `lint`: Ejecuta ESLint para analizar el código.
- `preview`: Previsualiza la versión de producción.

## Instalación

1. Clonar el repositorio:

   ```bash
   git clone https://github.com/tu-usuario/nombre-del-repo.git
   ```

2. Instalar las dependencias:
   ```bash
   cd nombre-del-repo
   npm install
   ```

## Uso

Para iniciar la aplicación en modo desarrollo:

```bash
npm run dev
```

Para crear una versión de producción:

```bash
npm run build
```

Para previsualizar la versión de producción:

```bash
npm run preview
```

Para desplegar la aplicación en GitHub Pages:

```bash
npm run deploy
```
